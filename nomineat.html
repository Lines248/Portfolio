NominEat

Full-Stack Group Voting Web App

NominEat is a full-stack web application designed to solve group indecision. Developed as part of a collaborative Agile team, the platform streamlines restaurant discovery by allowing users to create voting rooms and select a winner through a real-time platform.

  • JAVA
  • VUE.JS
  • POSTGRESQL
  • REST APIS
  • YELP API

The Challenge

Coordinating group plans often leads to friction and long decision times. My team's objective was to build a secure, decoupled system that could handle live data polling and real-time state changes for multiple users simultaneously.

The Solution

I served as a core developer on the team, focusing on the end-to-end architecture and API integration. I engineered the Java-based backend logic to facilitate restaurant discovery through the Yelp API and managed the data states in our PostgreSQL database. Users generate a unique link to invite friends to a voting session based on location and food preferences. Once the group polls are complete, the system calculates a winner and creates a formal event invite with the final details.
