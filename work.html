Own Your Data

Own Your Data is a simple prototype to demonstrate state persistence and basic blockchain-inspired update logic in the browser.

The goal was to build a small front-end tool where data doesn’t rely on a backend. I used localStorage to keep state persistent, and the update flow is modeled loosely after blockchain structure: each “change” becomes a new entry rather than overwriting the old one. It’s a straightforward way to show how a ledger-style system works using only HTML, CSS, and JavaScript.
  • Built with a modern stack React, Tailwind
  • Keyboard accessible UI controls
  • Optimized layout structure

Sea of Synapses

Sea of Synapses is a front-end visualization exploring how information flows through a network. The app uses vanilla JavaScript to generate nodes, group them by category, and draw connections dynamically based on shared attributes. It’s built for clarity, accessibility, and performance, with fully semantic structure and keyboard-friendly interaction patterns.

  • Built with vanilla JavaScript, semantic HTML, and modular CSS
  • Created a dynamic grouping + connection system using arrays & iteration
  • Designed an event-driven interaction model for hovering/focusing nodes
  • Modular functions to keep rendering and state clean
  • Optimized for smooth rendering and readability
