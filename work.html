Own Your Data

An interactive Next.js prototype exploring authorship and ownership through motion, sound, and accessible interaction. The one-way claim flow mirrors blockchain minting semantics in a lightweight local form.

  • Next.js 16, React 19, TypeScript
  • Tailwind CSS v4 + tokens
  • Framer Motion 12 for motion systems
  • Client-side persistence via localStorage
  • Accessible keyboard and reduced-motion support
  • Deployed on Vercel Edge

Sea of Synapses

Interactive data visualization prototype mapping relationships between cognitive states. Built with React, TypeScript, Tailwind, and Vite, withfully data-driven nodes and connection logic.

  • React + TypeScript
  • Tailwind CSS and CSS keyframe animation
  • Data-driven node system from nodes.json
  • Custom hooks for node lookup and category mapping
  • SVG neural pathways + animated glass-tile grid
  • Accessible focus states and reduced-motion support
